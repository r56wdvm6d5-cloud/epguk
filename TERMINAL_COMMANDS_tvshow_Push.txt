## Troubleshooting

### If Git Pull Fails (Divergent Branches)
```bash
# Option 1: Configure pull to use merge
git config pull.rebase false
git pull origin main

# Option 2: Configure pull to use rebase
git config pull.rebase true
git pull origin main

# Option 3: Use fast-forward only
git config pull.ff only
git pull origin main
```

### If Push Fails Due to Authentication
```bash
# Check git credentials
git config --list | grep user

# Set up credentials if needed
git config user.name "Your Name"
git config user.email "your.email@example.com"
```

### Force Push Variations
```bash
# Standard force push (can overwrite others' work)
git push --force origin main

# Safer force push (won't overwrite if remote has new commits)
git push --force-with-lease origin main
```
